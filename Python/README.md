## 📌 Giới thiệu
Các script Python được chia sẻ là những ví dụ phục vụ việc tự động hóa xử lý dữ liệu, báo cáo và phân tích. Mã nguồn chỉ thể hiện ý tưởng và quy trình thực hiện, không bao gồm toàn bộ dự án hoặc cấu trúc hoàn chỉnh. Một số đoạn code đã được rút gọn hoặc ẩn thông tin nhạy cảm, vì vậy chỉ mang tính chất tham khảo và học tập.
